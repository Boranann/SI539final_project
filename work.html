Projects

Coord: Cross-Time-Zone collaboration tool
December 2024

Coord is a collaboration tool for remote teams across time zones. It went through a full UX design process, including user research, ideation, prototyping, and usability testing. A high-fidelity interactive prototype is built using Figma.

Hear.Say Brewery UX Optimization Project
April 2025

Hear.Say Brewery is a UX optimization project for a local brewery in Ann Arbor. The project involved conducting user research and usability tests, providing actionable recommendations to improve the brewery's website and overall user experience.

My Travel Story: Interactive Map Web Application
April 2025

My Travel Story is an interactive map web application that allows users to pin locations on an interactive map and share personal travel stories. Built using HTML, CSS, JavaScript, and Leaflet.js, the application supports story submission, filtering by location, and persistent data storage via localStorage.

UMSI student success portal
March 2025

UMSI Student Success Portal is a client project for SI539. It was built using HTML, CSS, and JavaScript, with a focus on creating a responsive design for both phone and tablet views. The project features interactive UI components, accessibility best practices, and a clean, student-centered layout.

Heart Garden: Healing & Growth App for Breakups
November 2024

Heart Garden is a mobile app designed to help users navigate the emotional challenges of breakups. The app offers a range of features, including self-inprovement tasks, community support, and personalized therapists to assist users in their healing journey. Furthermore, market size, business model and social impact were analyzed.

AR Local guide
December 2024

AR Local guide aims to help foreign travelers experience cities like a local with the help of a knowledgeable AR guide. The project is more of a proof of concept developed in Unity to visualize the idea, rather than a fully developed product.
